La Telefonía IP es una tecnología que permite integrar en una misma red - basada en protocolo IP - las comunicaciones de voz y datos. En ocasiones se usa el término de redes convergentes o convergencia IP, lo cual integra (voz, datos, video, etc.).


Cuando hablamos de un sistema de telefonía IP estamos hablando de un conjunto de elementos que debidamente integrados permiten suministrar un servicio de telefonía (basado en VoIP) a la empresa. Los elementos básicos que forman este sistema son: la centralita IP(servidor IP), el Gateway IP y los diferentes teléfonos IP.
